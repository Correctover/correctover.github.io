The 5 Components

CCS-Core

Schema validation engine. Required/Supported/Forbidden fields.

Regulatory compliance. HIPAA, GDPR, SOC2 mapping.

CCS-Cost-Flow

Token budget enforcement. Prevents cost overruns.

CCS-Integrity

HMAC-based output binding. Cryptographic proof.

Comprehensive logging. Immutable audit trails.

Quick Integration

from correctover import CCSValidator

# Initialize validator
validator = CCSValidator(
    required_fields=["decision", "confidence", "trace_id"],
    supported_fields=["reasoning", "metadata"],
    forbidden_fields=["pii", "credentials"]
)

# Validate LLM output
result = validator.validate(llm_output)
if result.is_valid:
    print("✓ Output is compliant")
else:
    print(f"✗ Validation failed: {result.errors}")
